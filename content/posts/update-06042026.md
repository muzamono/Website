It's been half a year since I last updated this site.

I have migrated the stack from being hosted on AWS to Vercel just for a little bit of simplicity.

So, to describe my website, it is a statically generated website built with Next.js and Tailwind CSS, hosted on Vercel with continuous deployment from GitHub, and content managed through TinaCMS (a Git-backed headless CMS that lets me edit posts and content online without touching code).

For my own reference in the future, the key concepts are:

* Next.js : the framework that builds the site
* GitHub : stores the code and content (TinaCMS commits directly here)
* Vercel : hosts the site and auto-deploys whenever GitHub gets new commits
* TinaCMS : the online editor that sits on top, writing changes back to GitHub

The flow is: editing in TinaCMS → commits to GitHub → Vercel detects the commit → site rebuilds and goes live, all automatically.
